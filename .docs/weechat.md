# WeeChat

## Connect to freenode

* `/server add freenode chat.freenode.net`
* `/connect freenode`
* `/msg NickServ IDENTIFY username password`
* `/join #some-channel`

## Switching buffers

* <kbd>Meta</kbd> + <kbd>Left</kbd>
* <kbd>Meta</kbd> + <kbd>Right</kbd>

## Scrolling

On OSX:

* <kbd>Fn</kbd> + <kbd>Up</kbd>
* <kbd>Fn</kbd> + <kbd>Down</kbd>
